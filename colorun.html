Beach Colour Run

Color Run adalah salah satu event unggulan di Jember tepatnya event Colour Run pertama yang diakan setiap tahunnya di pantai Watu Ulo Jember. Event ini bertujuan untuk mempromosikan tempat wisata di Jember dan menjadi salah satu daya tarik yang sangat amat besar. untuk tempat dan tanggal tidak pasti tetapi selalu diadakan sekitar bulan Oktober sampai Desember
